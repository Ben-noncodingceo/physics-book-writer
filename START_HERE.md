# 🎯 从这里开始

## 你需要做的只有 2 步：

### 第 1 步：进入项目目录

在终端中运行：

```bash
cd /path/to/physics-book-writer
```

把 `/path/to/physics-book-writer` 替换成你的实际项目路径。

### 第 2 步：运行自动部署

```bash
./scripts/auto-deploy.sh
```

**就这样！** 脚本会自动完成所有工作：
- ✅ 创建数据库表格
- ✅ 安装依赖
- ✅ 部署后端
- ✅ 部署前端

## 等待 3-5 分钟

部署完成后，你会看到：

```
🎉 部署完成！
📊 部署信息:
  ✓ 数据库表格: 已自动创建
  ✓ 后端 API: https://physicsbookwriter.workers.dev
  ✓ 前端应用: https://physics-book-writer.pages.dev
```

## 访问你的应用

打开浏览器，访问：
👉 https://physics-book-writer.pages.dev
