# 🚀 一键自动部署

## 重要：先确保在正确的目录

```bash
# 进入项目目录
cd /path/to/physics-book-writer

# 确认当前在项目根目录（应该能看到 package.json）
ls package.json
```

## 运行自动部署脚本

```bash
./scripts/auto-deploy.sh
```

## 这个脚本会自动完成：

✅ 1. 检查并安装 Wrangler CLI
✅ 2. 验证 Cloudflare 登录（如未登录会自动打开浏览器）
✅ 3. 安装所有项目依赖
✅ 4. **自动创建数据库表格**（无需手动操作）
✅ 5. 自动构建和部署后端
✅ 6. 测试后端 API
✅ 7. 自动构建和部署前端

## 部署完成后访问

- **前端应用**: https://physics-book-writer.pages.dev
- **后端 API**: https://physicsbookwriter.workers.dev

## 如果遇到问题

### 问题 1: 权限错误

```bash
chmod +x scripts/auto-deploy.sh
./scripts/auto-deploy.sh
```

### 问题 2: 不在正确的目录

确保你在包含 `package.json` 的项目根目录：

```bash
# 查找项目目录
cd ~
find . -name "physics-book-writer" -type d 2>/dev/null

# 进入找到的目录
cd /path/to/physics-book-writer
```

### 问题 3: Wrangler 需要登录

脚本会自动提示登录，按照提示操作即可。

## 就这么简单！

整个过程大约 3-5 分钟，完全自动化，无需手动输入任何命令！
